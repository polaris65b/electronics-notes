# 멀티미터 사용법

## 1. 한 줄 요약

멀티미터는 회로의 전압, 저항, 도통, 전류를 확인하는 기본 계측기이며, 측정 모드와 리드선 위치를 잘못 선택하면 회로나 멀티미터가 손상될 수 있다.

## 2. 왜 필요한가?

임베디드 하드웨어를 디버깅할 때 가장 먼저 확인해야 하는 것은 전원이 정상인지, GND가 연결되어 있는지, 단선이나 쇼트가 없는지이다. 멀티미터는 이런 기본 상태를 빠르게 확인하는 도구이다.

멀티미터를 사용할 수 있으면 다음과 같은 문제를 확인할 수 있다.

- 전원 레일에 원하는 전압이 들어오는지
- GND가 제대로 연결되어 있는지
- 커넥터나 케이블이 단선되었는지
- 회로가 쇼트되었는지
- 저항값이 예상 범위에 있는지
- 부하가 어느 정도 전류를 사용하는지
- 다이오드와 LED의 방향이 맞는지

## 3. 핵심 개념

### 리드선

일반적인 멀티미터에는 검은색 리드선과 빨간색 리드선이 있다.

```text
검은색 리드선: COM 단자
빨간색 리드선: 측정 항목에 맞는 V/ohm/mA/A 단자
```

전압, 저항, 도통, 다이오드 측정은 보통 빨간색 리드선을 `V/ohm` 단자에 꽂는다. 전류 측정은 멀티미터에 따라 `mA` 또는 `A` 단자로 옮겨야 한다.

전류 측정 후 빨간색 리드선을 `V/ohm` 단자로 되돌리지 않으면, 다음 전압 측정 때 회로를 쇼트시킬 수 있다.

### 측정 모드

멀티미터는 같은 리드선을 사용하더라도 선택한 모드에 따라 내부 연결 방식이 달라진다.

| 모드 | 용도 | 회로 상태 |
| --- | --- | --- |
| DC Voltage | 직류 전압 측정 | 전원 인가 상태 |
| AC Voltage | 교류 전압 측정 | 전원 인가 상태 |
| Resistance | 저항값 측정 | 전원 차단 상태 |
| Continuity | 도통 확인 | 전원 차단 상태 |
| Diode | 다이오드 순방향 전압 확인 | 전원 차단 상태 |
| Current | 전류 측정 | 회로를 끊고 직렬 연결 |

저항, 도통, 다이오드 측정은 멀티미터가 자체적으로 작은 전압이나 전류를 흘려 확인한다. 따라서 회로에 전원이 들어온 상태에서 사용하면 잘못된 값이 나오거나 멀티미터가 손상될 수 있다.

### 병렬 측정과 직렬 측정

전압은 두 지점 사이의 차이를 보는 것이므로 측정 대상과 병렬로 연결한다.

```text
전압 측정:

+V ---- 부하 ---- GND
 |              |
 빨간 리드      검은 리드
```

전류는 흐르는 양을 보는 것이므로 회로를 끊고 그 사이에 멀티미터를 직렬로 넣는다.

```text
전류 측정:

+V ---- 멀티미터 ---- 부하 ---- GND
```

전류 측정 모드의 멀티미터는 내부 저항이 매우 낮다. 전류 모드로 전원과 GND를 직접 찍으면 쇼트가 발생한다.

## 4. 전압 측정

### DC 전압

배터리, 어댑터 출력, MCU 전원, 센서 전원처럼 직류 회로를 확인할 때 사용한다.

```text
1. 검은색 리드선을 COM에 꽂는다.
2. 빨간색 리드선을 V/ohm 단자에 꽂는다.
3. 모드를 DC Voltage로 선택한다.
4. 검은색 리드선을 기준점, 보통 GND에 댄다.
5. 빨간색 리드선을 측정하려는 지점에 댄다.
6. 표시값과 극성을 확인한다.
```

표시값이 음수라면 빨간색 리드선이 검은색 리드선보다 낮은 전위에 있다는 뜻이다. 배터리 극성을 반대로 찍었거나 기준점을 다르게 잡은 경우가 많다.

### 전원 확인 순서

전원 문제를 볼 때는 한 지점만 보지 말고 전원 경로를 따라가며 확인한다.

```text
1. 전원 장치 출력 단자
2. 보드 입력 커넥터
3. 보호 회로 뒤쪽
4. 레귤레이터 입력
5. 레귤레이터 출력
6. 실제 IC 전원 핀 근처
```

전원 장치에서는 정상 전압이 보여도 케이블, 커넥터, 다이오드, 퓨즈, PCB 패턴에서 전압이 떨어질 수 있다. 부하가 실제로 동작하는 상태에서 측정해야 한다.

### AC 전압

콘센트나 변압기 2차 측처럼 교류 전압을 확인할 때 사용한다. 상용 AC 전원은 감전 위험이 크므로 초보자는 직접 측정하지 않는 것이 좋다.

AC 전압을 측정할 때 DC Voltage 모드를 사용하면 값이 제대로 표시되지 않을 수 있다. 반대로 DC 회로를 AC 모드로 보면 리플이나 노이즈 일부만 보일 수 있다.

## 5. 저항과 도통 측정

### 저항 측정

저항 측정은 반드시 회로 전원을 끄고 수행한다.

```text
1. 회로 전원을 끈다.
2. 커패시터가 충전되어 있을 수 있으면 방전 상태를 확인한다.
3. 모드를 Resistance로 선택한다.
4. 측정할 부품 양단에 리드선을 댄다.
5. 표시값을 읽는다.
```

부품이 회로에 연결된 상태에서는 주변 부품이 병렬 경로를 만들 수 있어 실제 부품값과 다르게 측정될 수 있다. 정확한 저항값이 필요하면 한쪽 단자를 회로에서 분리하고 측정한다.

### 도통 측정

도통 모드는 두 지점이 낮은 저항으로 연결되어 있는지 빠르게 확인하는 기능이다. 보통 연결되어 있으면 삐 소리가 난다.

자주 쓰는 용도는 다음과 같다.

- 케이블 단선 확인
- 커넥터 핀 연결 확인
- GND끼리 연결되어 있는지 확인
- 전원과 GND가 쇼트되어 있는지 확인
- 납땜 브리지 확인

도통음이 난다고 해서 저항이 정확히 0ohm이라는 뜻은 아니다. 멀티미터마다 도통음이 나는 저항 기준이 다르다.

## 6. 다이오드 모드

다이오드 모드는 다이오드나 LED가 순방향으로 연결되었는지 확인할 때 사용한다. 표시값은 대개 순방향 전압 강하이다.

```text
빨간 리드: Anode
검은 리드: Cathode
```

일반적인 실리콘 다이오드는 순방향에서 대략 0.6~0.8V 정도가 표시될 수 있다. 쇼트키 다이오드는 더 낮게, LED는 색상에 따라 더 높게 표시될 수 있다.

역방향으로 측정했을 때는 보통 `OL` 또는 측정 범위 초과로 표시된다. 양방향 모두 매우 낮은 값이 나오면 쇼트, 양방향 모두 `OL`이면 단선이나 측정 조건 불일치를 의심할 수 있다.

## 7. 전류 측정

전류 측정은 멀티미터 사용 중 가장 실수하기 쉬운 항목이다. 전압처럼 병렬로 찍는 것이 아니라 회로를 끊고 직렬로 넣어야 한다.

```text
1. 회로 전원을 끈다.
2. 빨간색 리드선을 mA 또는 A 단자로 옮긴다.
3. 예상 전류가 크면 A 단자와 큰 전류 범위부터 시작한다.
4. 회로를 끊고 멀티미터를 직렬로 연결한다.
5. 회로 전원을 켠다.
6. 전류값을 읽는다.
7. 측정 후 전원을 끄고 리드선을 V/ohm 단자로 되돌린다.
```

전류가 어느 정도인지 모를 때는 가장 큰 전류 범위에서 시작한다. 작은 범위부터 시작하면 멀티미터 내부 퓨즈가 끊어질 수 있다.

부하 전류가 빠르게 변하는 경우 멀티미터 표시값만으로는 순간 피크를 보기 어렵다. 모터 기동 전류, 무선 송신 피크, MCU 슬립 전류처럼 시간 변화가 중요한 경우에는 오실로스코프와 전류 프로브 또는 션트 저항 측정을 검토한다.

## 8. 실무에서 자주 쓰는 측정 패턴

### 보드 전원이 안 켜질 때

```text
1. 전원을 끄고 VCC-GND 사이 저항이나 도통을 확인한다.
2. 쇼트가 없으면 전원을 켠다.
3. 입력 전압을 보드 커넥터에서 측정한다.
4. 레귤레이터 입력과 출력을 측정한다.
5. MCU나 주요 IC 전원 핀 근처에서 전압을 측정한다.
6. 부하 동작 순간에 전압이 떨어지는지 확인한다.
```

### 통신이 안 될 때

멀티미터만으로 UART, I2C, SPI 파형을 정확히 볼 수는 없다. 그래도 다음 기본 상태는 확인할 수 있다.

- GND가 서로 연결되어 있는지
- 신호선이 단선되지 않았는지
- 풀업 전압이 존재하는지
- TX/RX가 반대로 연결되지 않았는지
- 신호선이 전원이나 GND에 쇼트되지 않았는지

파형, 타이밍, 노이즈는 오실로스코프나 로직 애널라이저로 확인해야 한다.

### 전압 강하를 찾을 때

전압 강하는 의심되는 구간 양끝을 직접 찍어 찾는다.

```text
전원선 전압 강하 = 전원 쪽 +단자 전압 - 부하 쪽 +단자 전압
GND선 전압 강하 = 부하 쪽 GND 전압 - 전원 쪽 GND 전압
```

전압 강하는 부하 전류가 흐를 때 나타나므로 반드시 실제 동작 상태에서 측정한다.

## 9. 자주 하는 오해

- 전압 측정처럼 전류 측정을 병렬로 하면 안 된다.
- 전류 측정 후 리드선을 원래 단자로 되돌리지 않으면 다음 측정 때 쇼트가 날 수 있다.
- 저항과 도통 측정은 전원이 꺼진 상태에서 해야 한다.
- 도통음이 난다고 해서 완전한 0ohm 연결이라는 뜻은 아니다.
- 회로에 실장된 저항은 주변 경로 때문에 표시값이 실제 부품값과 다를 수 있다.
- 멀티미터는 빠른 순간 전압 강하나 노이즈를 잘 보여주지 못한다.
- 자동 범위 멀티미터라도 측정 모드와 리드선 위치는 사용자가 맞춰야 한다.
- GND를 아무 곳이나 기준으로 잡아도 항상 같은 값이 나오는 것은 아니다.

## 10. 실무 체크리스트

- [ ] 측정하려는 값이 전압, 저항, 도통, 전류 중 무엇인지 먼저 정했는가?
- [ ] 빨간색 리드선이 올바른 단자에 꽂혀 있는가?
- [ ] 전압 측정은 병렬, 전류 측정은 직렬이라는 점을 확인했는가?
- [ ] 저항, 도통, 다이오드 측정 전에 회로 전원을 껐는가?
- [ ] 전류 측정은 큰 범위에서 시작했는가?
- [ ] 전류 측정 후 빨간색 리드선을 V/ohm 단자로 되돌렸는가?
- [ ] 부하가 동작하는 상태에서 전압을 측정했는가?
- [ ] 기준 GND를 어디로 잡았는지 기록했는가?
- [ ] 순간적인 문제는 오실로스코프가 필요한지 판단했는가?

## 11. 정리

멀티미터는 하드웨어 디버깅의 출발점이다. 전압은 병렬로, 전류는 직렬로, 저항과 도통은 전원을 끄고 측정한다는 원칙을 지키면 대부분의 기본 점검을 안전하게 수행할 수 있다.

가장 중요한 습관은 측정 전마다 **모드, 리드선 위치, 측정 대상의 전원 상태**를 확인하는 것이다. 특히 전류 측정 모드에서 전압을 찍는 실수는 회로와 멀티미터를 손상시킬 수 있으므로 항상 주의해야 한다.
