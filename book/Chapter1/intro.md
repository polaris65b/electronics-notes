# Chatper 1 전자 공학 소개

이 책은 어디부터 시작하면 좋을까?라는 질문에 적합한 책이다.

순서는 이론 -> 실무에서 사용하는 지식들에 대한 소개 순으로 진행된다.


## 전원
- 배터리
- 직류 전원 공급 장치
- 교류 콘센트 ... 

## 검사 장비
- 오실로스코프
- 멀티미터
- 논리 탐촉자
- 함수 발생기
- 주파수 칸운터 ...

## 이산 피동 부품
- 저항기
- 커패시터
- 변압기
- 인덕터
- 결정체

## 기본 피동 회로
- 전류 및 전압 분배기
- 교류기 RC 지연회로
- 필터 ...

## 회로 구성
- 시제품 => 납땜 + 인두 -> PC 보드 => 외함

## 입력 장치
- 스위치
- 키보드
- 마이크
- 광트랜지스터
- 서미스터
- 변형계
- 발생기
- 수신 안테나

## 디지털 회로
- 논리 게이트(AND, NOT, XOR, ...)
- 플립플롭
- DRAM
- 카운터, 타이머, 프로세서, 교대 저항기, ...

## 사용처
- 컴퓨터
- 로봇
- 귀 체온계
- 자료 기록기, ...


## 아날로그 장치
연속적으로 변하는 전압이나 전류 신호를 다루는 장치이다.  
예를 들어 증폭기, 전압 조정기, 센서 신호 처리 회로 등이 있다.

## 디지털 전자기기
정해진 몇 가지 상태, 보통 고전압과 저전압의 두 상태로 정보를 표현하고 처리하는 장치이다.  
두 상태를 사용하면 데이터를 저장, 판단, 연산하기 쉽기 때문에 컴퓨터나 프로세서 같은 장치에 사용된다.
