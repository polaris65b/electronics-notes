# electronics-notes
전자회로와 하드웨어 인터페이스에 대한 이해를 쌓기 위해 정리하는 학습 저장소입니다.

이 저장소는 회사 내부 자료나 특정 제품 정보를 기록하기 위한 공간이 아닙니다.
공개 자료, 교재, 데이터시트, 개인 학습 내용을 바탕으로 전자 기초 개념과 임베디드 실무에서 자주 마주치는 내용을 정리합니다.

---

## 목적

이 저장소의 목적은 다음과 같습니다.

1. 전자회로 기초 개념을 체계적으로 정리한다.
2. 임베디드 개발자가 하드웨어를 이해하는 데 필요한 실무 지식을 쌓는다.
3. 데이터시트를 읽고 회로 동작을 해석하는 능력을 기른다.
4. MCU, 센서, 전원, 통신, 디버깅 관련 지식을 누적한다.
5. 학습 과정에서 헷갈렸던 점과 실수했던 부분을 기록하여 반복 학습한다.

### 3. 데이터시트를 우선한다

부품 관련 내용은 블로그나 영상보다 데이터시트를 우선합니다.

특히 다음 항목은 반드시 데이터시트에서 확인합니다.

* Absolute Maximum Ratings
* Recommended Operating Conditions
* Electrical Characteristics
* Input / Output Voltage Level
* Current Rating
* Power Dissipation
* Thermal Resistance
* Switching Characteristics
* Package Information
* Application Circuit
* Layout Guideline

---

### 4. 계산 과정은 생략하지 않는다

저항값, 전류, 전력, 발열, 주파수, RC 시정수 등을 계산할 때는 결과만 적지 않고 계산 과정을 함께 남깁니다.

예시:

```text
LED 전류 제한 저항 계산

Vcc = 3.3V
Vf = 2.0V
I = 10mA

R = (Vcc - Vf) / I
R = (3.3V - 2.0V) / 0.01A
R = 130Ω

따라서 표준 저항값으로 150Ω 또는 180Ω을 검토할 수 있다.
```
