# 트랜지스터 실무 사용 정리

## 1. 한 줄 요약

트랜지스터를 실무에서 사용할 때는 부하 전류, 구동 전압, 발열, 보호 소자, 데이터시트 조건을 함께 확인해야 한다.

## 2. 기본 설계 순서

트랜지스터 회로를 설계할 때는 다음 순서로 확인한다.

1. 어떤 부하를 켜고 끌 것인지 정한다.
2. 부하 전압과 부하 전류를 확인한다.
3. 로우사이드 또는 하이사이드 스위치 구조를 선택한다.
4. BJT 또는 MOSFET 중 적절한 소자를 고른다.
5. 제어 신호의 전압을 확인한다.
6. 베이스 저항 또는 게이트 주변 저항을 정한다.
7. 전력 손실과 발열을 계산한다.
8. 릴레이, 모터, 솔레노이드라면 플라이백 다이오드를 추가한다.
9. 데이터시트의 정격과 실제 사용 조건을 비교한다.

## 3. BJT를 스위치로 사용할 때

### 회로 형태

NPN BJT 로우사이드 스위치는 다음과 같은 형태로 많이 사용한다.

```text
+Vcc
 |
 부하
 |
 Collector
   NPN BJT
 Emitter
 |
 GND

MCU GPIO -- Rb -- Base
```

GPIO가 HIGH가 되면 베이스 전류가 흐르고 BJT가 켜진다. 부하 전류는 Collector에서 Emitter로 흐른다.

### 베이스 저항 계산

BJT를 스위치로 사용할 때는 포화 영역에 들어가도록 베이스 전류를 충분히 흘려야 한다.

예를 들어 부하 전류가 100mA이고 강제 전류 증폭률을 10으로 잡으면 필요한 베이스 전류는 다음과 같다.

```text
Ic = 100mA
강제 beta = 10

Ib = Ic / 10
Ib = 100mA / 10
Ib = 10mA
```

GPIO가 3.3V이고 베이스-이미터 전압을 0.7V로 보면 베이스 저항은 다음과 같다.

```text
Rb = (Vgpio - Vbe) / Ib
Rb = (3.3V - 0.7V) / 0.01A
Rb = 260 ohm
```

실제 설계에서는 GPIO가 10mA를 안정적으로 공급해도 되는지 먼저 확인해야 한다. GPIO 전류가 부담되면 MOSFET 사용을 검토한다.

### BJT 사용 시 체크리스트

- [ ] 부하 전류가 BJT의 Ic 정격보다 충분히 작은가?
- [ ] Vce(max)가 회로 전압보다 충분히 높은가?
- [ ] 베이스 저항을 넣었는가?
- [ ] GPIO가 필요한 베이스 전류를 공급할 수 있는가?
- [ ] Vce(sat)로 인한 전력 손실을 계산했는가?
- [ ] 코일 부하라면 플라이백 다이오드를 넣었는가?

## 4. MOSFET을 스위치로 사용할 때

### 회로 형태

N채널 MOSFET 로우사이드 스위치는 다음과 같은 형태로 많이 사용한다.

```text
+Vcc
 |
 부하
 |
 Drain
   N-MOSFET
 Source
 |
 GND

MCU GPIO -- Rg -- Gate
Gate -- Rpd -- GND
```

게이트 직렬 저항 Rg는 스위칭 순간의 충전 전류와 노이즈를 완화한다. 게이트 풀다운 저항 Rpd는 MCU 핀이 리셋 중이거나 입력 상태일 때 MOSFET이 의도치 않게 켜지는 것을 막는다.

### 로직 레벨 MOSFET 확인

3.3V MCU로 MOSFET을 구동할 때는 다음을 확인한다.

- Rds(on)이 Vgs = 2.5V 또는 3.3V 근처에서 보장되는가?
- 부하 전류에서 전력 손실이 허용 가능한가?
- 패키지가 발생 열을 감당할 수 있는가?

Vgs(th)만 보면 안 된다. Vgs(th)는 MOSFET이 본격적으로 켜지는 전압이 아니라, 아주 작은 전류가 흐르기 시작하는 조건이다.

### MOSFET 전력 손실 계산

MOSFET이 켜졌을 때의 주요 손실은 다음처럼 계산한다.

```text
P = I^2 * Rds(on)
```

예를 들어 부하 전류가 2A이고 Rds(on)이 50mOhm이면 다음과 같다.

```text
P = 2A * 2A * 0.05ohm
P = 0.2W
```

0.2W가 작아 보이더라도 패키지와 PCB 방열 조건에 따라 온도 상승이 문제가 될 수 있다.

### MOSFET 사용 시 체크리스트

- [ ] Vds(max)가 회로 전압보다 충분히 높은가?
- [ ] Id(max)가 부하 전류보다 충분히 높은가?
- [ ] Rds(on)이 실제 Vgs 조건에서 보장되는가?
- [ ] I^2 * Rds(on)으로 전력 손실을 계산했는가?
- [ ] 게이트 풀다운 저항을 넣었는가?
- [ ] 빠른 스위칭이 필요하면 게이트 전하와 스위칭 손실을 확인했는가?
- [ ] 코일 부하라면 플라이백 다이오드나 스너버를 검토했는가?

## 5. 릴레이 구동 예제

5V 릴레이 코일 전류가 70mA이고 MCU GPIO가 3.3V라고 가정한다.

### BJT로 구동

```text
릴레이 코일 전류 = 70mA
강제 beta = 10

필요 베이스 전류 = 70mA / 10
필요 베이스 전류 = 7mA
```

베이스 저항은 다음과 같다.

```text
Rb = (3.3V - 0.7V) / 0.007A
Rb = 약 371 ohm
```

표준 저항값으로 330ohm 또는 390ohm을 검토할 수 있다. 단, GPIO가 7mA를 허용하는지 확인해야 한다.

릴레이 코일 양단에는 플라이백 다이오드를 병렬로 넣는다.

### MOSFET으로 구동

릴레이 전류가 70mA라면 작은 로직 레벨 N-MOSFET으로도 충분한 경우가 많다.

확인할 항목은 다음과 같다.

- Vds(max)가 5V보다 충분히 높은가?
- Id(max)가 70mA보다 충분히 높은가?
- Vgs = 3.3V에서 Rds(on)이 충분히 낮은가?
- 게이트 풀다운 저항을 넣었는가?
- 플라이백 다이오드를 넣었는가?

## 6. LED 구동 예제

LED 하나를 GPIO로 직접 켤 수 있는 경우도 있지만, 전류가 크거나 여러 개를 동시에 켜야 하면 트랜지스터를 사용한다.

```text
+Vcc -> LED -> 전류 제한 저항 -> 트랜지스터 -> GND
```

LED에는 반드시 전류 제한 저항이 필요하다. 트랜지스터는 LED 전류를 대신 흘려 주는 스위치 역할을 할 뿐, LED 전류를 자동으로 제한해 주지 않는다.

## 7. 모터 구동 시 주의점

DC 모터는 정격 전류보다 기동 전류와 정지 전류가 훨씬 클 수 있다. 데이터시트나 실측으로 피크 전류를 확인해야 한다.

주의할 점은 다음과 같다.

- 모터 기동 전류를 고려한다.
- MOSFET의 순간 전류 정격과 발열을 확인한다.
- 플라이백 다이오드나 TVS 다이오드 등 보호 소자를 검토한다.
- PWM 제어를 한다면 스위칭 손실과 게이트 구동 능력을 확인한다.
- 모터 전류가 MCU 전원에 노이즈를 만들 수 있으므로 전원 분리와 디커플링을 검토한다.

## 8. 실무 디버깅 포인트

트랜지스터 회로가 기대대로 동작하지 않을 때는 다음을 확인한다.

- GPIO가 실제로 HIGH 또는 LOW로 바뀌는가?
- BJT의 Base, Collector, Emitter 방향이 맞는가?
- MOSFET의 Gate, Drain, Source 방향이 맞는가?
- GND가 공통으로 연결되어 있는가?
- 부하 전원과 MCU 전원이 의도한 기준점을 공유하는가?
- 베이스 저항값이 너무 크거나 작은가?
- MOSFET이 로직 레벨 타입인가?
- 코일 부하에 플라이백 다이오드 방향이 맞는가?
- 부하 전류가 예상보다 크지 않은가?
- 소자가 뜨겁다면 전력 손실 계산과 패키지 방열 조건이 맞는가?

## 9. 정리

BJT는 간단한 저전류 스위칭과 기본 개념 학습에 좋다. MOSFET은 GPIO 전류 부담을 줄이고 고전류 부하를 효율적으로 제어하기 좋다.

실무에서는 회로가 켜지고 꺼지는지만 보지 말고, 데이터시트 조건에서 충분히 켜지는지, 전류와 전압 정격에 여유가 있는지, 발열과 역기전력에 대한 보호가 되어 있는지 확인해야 한다.
